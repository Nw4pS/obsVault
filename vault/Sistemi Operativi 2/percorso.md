il percorso è il cammino dal root fino alla cartella o file su cui si sta lavorando, ad es di **percorso assoluto**:
```Bash
obsidian@debian:/home/Downloads/BadApple.mp3$
```
la cartella home viene anche indicata come ~ detta "tilde", quindi è valida anche:
```Bash
obsidian@debian:~/Downloads/BadApple.mp3$
```
