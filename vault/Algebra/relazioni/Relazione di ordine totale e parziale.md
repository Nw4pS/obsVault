# relazione d'ordine totale
una relazione d'ordine totale è una relazione dove valgono le seguenti proprietà:
- riflessività: $\forall x \in X\space x\lt x$ 
- anti-simmetria: $\forall x, y \in X; \space x\lt y\text{ e } y\lt x \implies x=y$ 
- transitività
- totalità c'è sempre un valore minore di un valore minore di un altro (se stesso per esempio)
# relazione d'ordine parziale
se non rispetta la totalità
